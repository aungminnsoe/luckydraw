# 🎱 Lucky Draw

A mobile-friendly Singapore TOTO number generator with daily horoscope, lucky numbers, and TOTO result checker.

## Features

- 🌅 **Daily luck score** based on your birth date
- ✨ **Western + Chinese zodiac** with daily lucky numbers
- 🎲 **5 number generation strategies** (Quick, Lucky, Hot, Cold, Balanced)
- 🎯 **Singapore TOTO results** with built-in history of last 20 draws
- ✅ **Check your numbers** against the latest draw
- 👤 **Multiple profiles** — switch between family members
- 💾 **Saves to your phone** — no account, no cloud
- 📱 **Installable as an app** (Add to Home Screen)

## Quick Setup on GitHub Pages

### 1. Create the repo
- Go to github.com → click **New repository**
- Name it `lucky-draw` (or whatever you like)
- Make it **Public**
- Don't add README/license yet (we're uploading our own)
- Click **Create repository**

### 2. Upload these files
On the empty repo page, click **uploading an existing file** and drag in:
- `index.html`
- `manifest.json`
- `icon.svg`

Click **Commit changes**.

### 3. Enable GitHub Pages
- Go to **Settings** (top-right of the repo)
- Click **Pages** (left sidebar)
- Under **Source**, pick **Deploy from a branch**
- Branch: `main`, folder: `/ (root)`
- Click **Save**

Wait 1-2 minutes. Your app will be live at:

```
https://YOUR-USERNAME.github.io/lucky-draw/
```

### 4. Add to home screen
- Open the URL in **Safari** (iPhone) or **Chrome** (Android)
- Tap the **Share** button (iPhone) or three-dot menu (Android)
- Tap **Add to Home Screen**
- Done — opens like a real app, fullscreen, no browser bars!

### 5. Share with your wife
Just send her the URL. She opens it on her phone, adds to home screen, sets up her own profile. Her data stays on **her phone** — completely separate from yours.

## Updating TOTO Results

The app comes bundled with the last 20 draws. After every new draw (Mon/Thu 6:30pm), tap the **+ Add Draw** button in the TOTO tab to add the latest numbers. The app remembers them locally.

## Notes

- All data is stored in your phone's local storage. Nothing is sent anywhere.
- Lottery outcomes are random — hot/cold analysis is for entertainment only.
- Play responsibly. 🍀
